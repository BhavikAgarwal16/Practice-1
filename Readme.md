# Superstore Data Analysis in Power BI

This repository contains a data analysis project centered around the **Superstore dataset**, designed and developed using **Microsoft Power BI**. The goal is to explore sales performance, customer behavior, and operational insights across various regions and product categories.

## 🔍 Project Overview

- **Dataset:** Superstore sales data (CSV or Excel format). Includes order details such as date, product, category, sales, profit, quantity, and region.
- **Dashboard:** A Power BI `.pbix` file encapsulating visualizations, measures, and interactive reports.
- **Objective:** Provide business intelligence insights that can help stakeholders make data-driven decisions about inventory, marketing, and operations.

## 📂 Repository Structure

```
/ (root)
├── Superstore_Data.xlsx       # or CSV file containing the source data
├── Superstore_Dashboard.pbix  # Power BI dashboard with reports
└── Readme.md                  # Project documentation (this file)
```

> ⚠️ Ensure the data file and the dashboard file are kept together. Opening the PBIX without the data source may require relinking or refreshing.

## 🚀 Getting Started

1. **Clone the repository** to your local machine:
   ```bash
   git clone <repo-url>
   ```
2. **Open Power BI Desktop.**
3. **Load the dataset:**
   - If the dashboard doesn't automatically connect, import the `Superstore_Data.xlsx` or `CSV` into Power BI.
4. **Review the dashboard:**
   - Navigate through pages to view sales trends, regional performance, product category breakdowns, and customer segments.
   - Use filters and slicers to interact with the visualizations.
5. **Refresh the data** if you update the dataset.

## 📊 Key Features of the Dashboard

- **Sales Overview:** Total sales and profit metrics with time-series charts.
- **Regional Analysis:** Map visuals and bar charts detailing performance by region and state.
- **Product Insights:** Category-wise breakdown and top-selling items.
- **Customer Behavior:** Segment analysis and order frequency.
- **Profitability:** Visuals highlighting high and low margin areas.

## 🛠️ Tools & Technologies

- **Microsoft Power BI Desktop** for report creation and visualization
- **Excel/CSV** for raw data storage
